`git clone https://github.com/kitravee/kitcamp-future-template.git`\
⚡️ go to project\
`cd kitcamp-future-template`\
⚡️ double click\
`index.html`

### Header Section 😊

⚡️ Change Image background (For mobile device): go to `.\assets\img` and change file header-bg (.jpg)
